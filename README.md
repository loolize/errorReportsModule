# errorReportsModule
Модуль автоматически фиксирует ошибки и формирует структурированные отчеты

1. Подготовка модуля к использованию
Перед тем как начать работу, необходимо убедиться, что файл error_module.py модуля находится в каталоге проекта. 
2. Активация модуля
Чтобы модуль начал работу, его необходимо активировать. Это делается в самом начале программы.
import error_module
…
error_module.activate_error_reporting()

Если модуль активирован успешно, в консоли появится короткое сообщение: «Модуль отчетов об ошибках активирован. Каталог логов: …»
Каталог logs создается автоматически при первом возникновении ошибки или вручную при активации. Это место, куда будут записываться отчеты.

3.	 Настройка каталога для отчетов
Папку для логов можно указать вручную:
error_module.activate_error_reporting(log_dir="my_errors")

В этом случае все отчеты будут сохранены в каталог my_errors.
